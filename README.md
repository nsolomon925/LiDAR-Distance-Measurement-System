# LiDAR-Distance-Measurement-System
A portable, battery-powered distance measurement solution designed for precision and real-time feedback. This project integrates an ATmega 2560 microcontroller with a TF-Luna LiDAR sensor using the I2C communication protocol to provide high-frequency distance data.

## 🚀 Key Features
* **Real-Time Processing:** High-speed data acquisition from the TF-Luna sensor.
* **Visual Output:** Integrated **1602 LCD** for clear, instant distance readouts.
* **Rugged Portability:** Housed in a durable ammo box casing, making it suitable for field testing.
* **Optimized Power:** Designed as a standalone, battery-powered unit for mobile use.

## 🛠️ Technical Specifications
* **Microcontroller:** ATmega 2560.
* **Sensor:** TF-Luna LiDAR (via I2C communication).
* **Display:** 1602 LCD with I2C integration.
* **Software:** Custom logic for sensor calibration and data smoothing.
